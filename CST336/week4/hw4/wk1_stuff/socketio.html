SOCKET IO

Socket IO is a multi-language library that provides full-duplex, low-latency communication between servers and client programs. It makes it easy to keep persisten TCP connections for communication. Visit Socket.io!

  • Performance

    SocketIO is a light wrapper for WebSocket, which means it makes using sockets easier with little added overhead.

  • Reliable

    If socket connection is not possible due to firewall or other reasons, SocketIO will fallback to HTTP long-polling to ensure the job is done.

  • Scalable

    SocketIO can be spread across multiple servers and handle as many connected clients as hardware will permit.
