MONGOOSE JS

MongooseJS is a javascript library that allows you to easily connect your web app to a MongoDB database. Visit MongooseJS!

  • NoSQL Database

    MongoDB is a schemaless database, allowing your data to take whatever shape it needs to be.

  • Asynchronous

    MongooseJS supports the use of promises and callbacks, allowing you to handle your data processing with elegance and ease.

  • Easy To Use

    Mongoose integrates perfectly with NodeJS and ExpressJS to abstract away the complexity of javascript database access. It makes it easy!
